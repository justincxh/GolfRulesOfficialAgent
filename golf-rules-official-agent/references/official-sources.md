text into a public repository without permission from the rightsholder.

## Source Hierarchy

For current rulings, use current Rules, Definitions, Clarifications, Additional
Clarifications, Committee Procedures, Model Local Rules, and current Official
Guide materials. Do not use old Decisions or old rulebooks as current authority.

Use old Decisions mapping only when a user gives an old Decision number or asks
about historical rules.
